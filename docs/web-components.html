StreamGrid — Web Component Demo

Declarative HTML-first usage with <stream-grid> and <stream-grid-column>. No JavaScript required for the basic case — every grid on this page is configured entirely in HTML attributes.

Zero JS Required Custom Elements v1 No Build Step

1. Minimal — Auto-Discovered Columns

Point src at a REST API and set a table. StreamGrid auto-discovers columns from the first row of data. No <stream-grid-column> children needed.

<stream-grid static-src="./demo-data.json" table="users" page-size="5"> </stream-grid>

2. Explicit Columns with Sorting

Declare columns as <stream-grid-column> children to control field order, labels, widths, and sort types. The sorter attribute sets the comparator ("string", "number", or "date").

<stream-grid src="…" table="users" page-size="5" pagination-mode="numbers"> <stream-grid-column field="id" label="ID" width="60px" sorter="number" sortable="false"></stream-grid-column> <stream-grid-column field="name" label="Full Name" sorter="string"></stream-grid-column> <stream-grid-column field="email" label="Email" sortable="false"></stream-grid-column> <!-- … more columns --> </stream-grid>

3. Declarative Filtering

Add the boolean filter attribute to any column to include it in the filter. Set filter-mode and filter-debounce on the parent element to control filtering strategy and input responsiveness.

<stream-grid src="…" filter-mode="client" filter-debounce="200"> <stream-grid-column field="name" label="Name" filter></stream-grid-column> <stream-grid-column field="email" label="Email" filter sortable="false"></stream-grid-column> <stream-grid-column field="department" label="Department" filter sorter="string"></stream-grid-column> <!-- columns without filter are searchable but don't appear in the filter field list --> </stream-grid>

4. Declarative Cell Rendering with <template>

Reference a <template> element by ID via the template attribute. Use {{value}} for the cell value and {{row.field}} for any other field in the row. All interpolated values are HTML-escaped automatically — no XSS risk.

<!-- Define templates once, reference by ID --> <template id="status-tpl"> <span class="badge badge-{{value}}">{{value}}</span> </template> <template id="profile-tpl"> <strong>{{value}}</strong> <small>({{row.department}})</small> </template> <!-- Reference via the template attribute --> <stream-grid-column field="name" template="profile-tpl"></stream-grid-column> <stream-grid-column field="status" template="status-tpl"></stream-grid-column>

5. Infinite Scroll

Set pagination-mode="infinite" for scroll-triggered row loading. No pagination controls are rendered — rows append automatically as the user scrolls.

<stream-grid src="…" pagination-mode="infinite" page-size="20"> <!-- columns … --> </stream-grid>

6. The element.grid Escape Hatch

Access the full StreamGrid instance via element.grid for programmatic control — event subscription, plugin registration, and render callbacks. This demo subscribes to sort and filter events and logs them below.

// Access the full StreamGrid instance after the element connects const el = document.querySelector('stream-grid'); el.grid.on('sortChanged', ({ sortStack }) => { console.log('Sort:', sortStack); }); el.grid.on('filterApplied', ({ filterText, totalFilteredRows }) => { console.log(`Filter: "${filterText}" → ${totalFilteredRows} rows`); });
